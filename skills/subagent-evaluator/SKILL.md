# subagent-evaluator Skill

> 用于评估和展示子Agent能力，自动选择合适的子Agent处理任务

## 触发条件

用户描述的任务涉及以下场景时自动触发：

- 需要搜索信息、查资料、浏览网页
- 需要学习新知识、研究某个主题
- 需要写代码、调试、技术任务

## 能力映射

| 子Agent | 专长 | 适用场景 |
|---------|------|----------|
| researcher | 搜索、信息处理、网页内容 | 查资料、做调研、了解新信息 |
| learner | 任务整合、学习规划 | 学习新东西、整理知识 |
| coder | 代码开发、自动化、调试 | 写代码、修Bug、技术实现 |

## 使用方法

### 1. 任务分发

根据用户需求，自动选择合适的子Agent：

```
研究类任务 → researcher
学习类任务 → learner  
技术类任务 → coder
```

### 2. 能力展示

用户询问"你能做什么"或"子Agent是什么"时，输出以上表格

## 输出格式

简洁清晰，用表格或列表展示
