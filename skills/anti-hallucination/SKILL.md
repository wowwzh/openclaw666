# Anti-Hallucination 反幻觉Prompt

使用这些模板让AI回答更可靠。

## 通用问答模板

```
请基于以下参考文档回答。
如果文档中没有明确答案，请明确说"不确定"。

参考文档：
{context}

要求：
1. 只基于提供的文档回答，不要编造信息
2. 回答格式：
   - 答案：[你的答案]
   - 来源：[引用的文档]
   - 置信度：高/中/低
3. 如果不确定，直接说"不确定"
```

## 推理任务模板

```
请分步骤思考这个问题：
{question}

步骤要求：
1. 理解问题 - 用自己的话复述
2. 分析已知信息 - 列出相关事实
3. 推导结论 - 逐步推理
4. 验证结论 - 检查逻辑是否自洽

最后给出答案和置信度评分。
```

## 代码/技术任务模板

```
任务：{task}

要求：
1. 只使用你确定的技术栈和API
2. 如果不确定某个API的用法，明确说明
3. 提供可运行的完整代码
4. 标注可能的风险点
```

## 专业领域模板

```
你是一个{领域}专家。

任务：{question}

要求：
1. 只基于{领域}专业知识回答
2. 引用可靠的来源或标准
3. 不确定的部分明确标注
4. 如需人工审核，标注"需专家确认"
```

## 多模型验证模板

```
请用多种方法解决这个问题：

方法1：{method1}
方法2：{method2}
方法3：{method3}

对比三种方法的结果：
- 一致的答案：可信度[高/中]
- 不一致的答案：需人工复核
```

## 使用方式

```
# 在回答问题前，先加载这个技能
使用反幻觉技能回答：{你的问题}
```

## 核心原则

1. **引用来源** - 标注信息来源
2. **标注置信度** - 不确定时说"不知道"
3. **分步推理** - 展示思考过程
4. **多层验证** - 必要时用多个方法交叉验证
