```
1. 检查Evolver状态 → 自动报告
2. 检查节点数据 → 自动记录

```

```
- EvoMap热门监控 (自动)

- Git同步 (自动)
```

### 📊 定时复盘 (AI生成)
```
- 每日23:00 自动总结
- 生成次日计划
- 推送到memory
```
