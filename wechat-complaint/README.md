# 电影票代购商 - 微信支付投诉自动处理系统

## 业务背景

**业务模式：**
- 客户通过公众号/小程序下单购买电影票
- 代购商负责订票，返回取票信息

**投诉类型和处理方式：**

### 1. 个人原因取消
- 关键词：不想要、没时间、个人原因、退票、退款
- 处理：**拒绝退款**
- 回复模板：抱歉，由于个人原因无法观影，按照平台规定，不支持退票退款

### 2. 取票码问题
- 关键词：取票码、没收到、取不了
- 处理：**查询订单**
  - 订单正常 → 告知取票码，拒绝退款
  - 订单有误 → **反馈给老板核实**

### 3. 订单问题
- 关键词：订单错误、订错了
- 处理：**反馈给老板**

### 4. 支付问题
- 关键词：支付失败、扣款了
- 处理：**紧急处理**

## 处理流程

```
1. 拉取投诉列表
       ↓
2. 分析投诉类型（关键词匹配）
       ↓
3. 根据类型处理
   ├── 个人原因 → 直接回复（拒绝退款）
   ├── 取票问题 → 查询订单
   │              ├── 正常 → 告知取票码
   │              └── 异常 → 反馈给老板
   └── 其他 → 人工处理
       ↓
4. 提交回复
       ↓
5. 生成处理报告
```
