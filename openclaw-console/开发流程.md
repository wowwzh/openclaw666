# OpenClaw Console 开发流程

## 我的角色：主控调度

作为主控，我需要：
- ✅ 主动规划任务
- ✅ 分配给对应Agent
- ✅ 追踪进度
- ✅ 汇总结果
- ✅ 安排下一步

## 多Agent协作流程

```
📚 researcher → 🎨 architect → 💻 coder → 🔍 reviewer → 🧪 tester → 🚀 deployer
```

| 阶段 | Agent | 任务 |
|------|-------|------|
| 调研 | researcher | 调研技术/方案 |
| 设计 | architect | 架构设计/文档 |
| 开发 | coder | 写代码实现 |
| 审查 | reviewer | 代码审查/找问题 |
| 测试 | tester | 写测试/验证 |
| 部署 | deployer | 打包/发布 |

---

## ⚠️ 交付前必检清单 (最重要！)

每次build后必须检查：
1. **dist/index.html 路径**：必须是相对路径 (`assets/xxx`，不能是 `/assets/xxx`)
2. **lang="zh-CN"**：必须是中文
3. **title**：必须是中文
4. **electron.js**：必须加载 `dist/index.html`，不能是 `localhost:5173`
5. **启动无白屏**

---

## 代码审查要点

必须检查：
1. **白屏相关**（最优先！）：index.html路径、语言、title、electron配置
2. 功能完整性
3. 类型安全
4. 性能问题
