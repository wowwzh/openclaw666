# Simple RAG - 检索增强生成实战项目

一个纯前端实现的简单RAG系统，用于学习和理解RAG的核心概念。

## 🌟 功能特性

1. **文档加载** - 预置8篇技术文档（JavaScript、Python、前端、后端、数据库、机器学习等）
2. **文档分块** - 将长文档分割成较小的语义块，支持重叠
3. **向量化** - 使用简单的TF-IDF词袋模型进行向量化
4. **向量存储** - 使用数组模拟向量存储
5. **相似度搜索** - 基于余弦相似度进行语义检索
6. **问答界面** - 友好的聊天界面，支持示例问题

## 🚀 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 等待系统初始化完成（状态栏显示"系统就绪"）
3. 输入问题或点击示例问题进行问答

1. **更复杂的向量化** - 可以集成 sentence-transformers.js 或使用 OpenAI 的 embedding API
