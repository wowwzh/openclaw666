## 一、日常使用技巧

### 1.1 快捷键提升效率

| 快捷键 | 功能 |
|--------|------|
| Win + D | 显示桌面 |
| Win + E | 打开文件资源管理器 |
| Win + L | 快速锁屏 |
| Win + Shift + S | 截图工具 |
| Ctrl + Shift + Esc | 直接打开任务管理器 |
| Alt + Tab | 切换窗口 |
| Win + V | 剪贴板历史记录 |

### 1.2 文件管理好习惯

- **定期整理桌面**：桌面文件过多会拖慢系统启动速度
- **合理分区**：建议系统盘(C盘)保留至少20%空闲空间
- **命名规范**：使用 `日期_项目_版本` 格式，便于检索
- **善用库功能**：将常用文件夹固定到快速访问

### 1.3 软件安装原则

- ✅ 优先使用微软商店或官网下载
- ✅ 安装时取消勾选"推荐软件"
- ❌ 避免安装多个同类软件（如多个杀毒软件）
- ❌ 谨慎使用破解软件，防止携带病毒

## 二、磁盘清理与存储优化

### 2.1 清理工具

| 工具 | 用途 |
|------|------|
| 磁盘清理 | 删除临时文件、系统缓存 |
| 存储感知 | 自动清理不需要的文件 |
| CCleaner | 第三方清理（谨慎使用） |

### 2.2 清理步骤

1. 打开"设置" → "存储"
2. 启用"存储感知"
3. 手动运行"磁盘清理"
4. 清理Windows更新缓存

### 2.3 大文件查找

```powershell
# 查找C盘大于100MB的文件
Get-ChildItem C:\ -Recurse -File -ErrorAction SilentlyContinue | 
  Where-Object { $_.Length -gt 100MB } | 
  Sort-Object Length -Descending | 
  Select-Object FullName, @{N='SizeMB';E={[math]::Round($_.Length/1MB,2)}} | 
  Select-Object -First 20
```

## 三、系统优化

### 3.1 开机启动项管理

```powershell
# 查看启动项
Get-StartupApps

# 或在任务管理器 → 启动 中禁用不需要的
```

### 3.2 电源设置

- 平衡模式：日常使用
- 高性能：游戏/专业软件
- 节能模式：外出办公

### 3.3 视觉效果

- 右键"此电脑" → "属性" → "高级系统设置" → "性能" → "调整为最佳性能"

## 四、常见问题排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 电脑卡顿 | 内存不足/磁盘满 | 清理磁盘/增加内存 |
| 开机慢 | 启动项过多 | 禁用启动项 |
| 蓝屏 | 驱动/硬件问题 | 安全模式排查 |
| 突然关机 | 过热/电源问题 | 检查散热/电源 |
| 文件丢失 | 误删/硬盘坏道 | 检查回收站/磁盘检测 |
