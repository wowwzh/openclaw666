# 🌅 早晨助手 - 整合方案

> 目标：把 1.新闻聚合 2.知识库 3.每日简报 整合成一个超级应用

---

## 📋 任务拆分

### Phase 1: 科技新闻聚合 (Tech News Digest)
**目标**: 实时获取科技/AI 新闻

**方案**:
- [ ] 接入真实新闻 API (用已有的 Tavily 搜索)
- [ ] 定时抓取科技热点
- [ ] 按分类筛选 (AI/商业/硬件等)

**技术栈**: 
- 后端: FastAPI + 定时任务
- 数据源: Tavily 搜索 / 今日头条 API

---

### Phase 2: 知识库增强 (Knowledge Base RAG)
**目标**: 构建真正的 RAG 知识库

**方案**:
- [ ] 支持上传 PDF/MD/TXT 文件
- [ ] 向量存储 (已有 local-vector-store)
- [ ] 语义搜索
- [ ] 自动摘要

**技术栈**:
- 向量库: local-vector-store
- 文件解析: python-docx, pypdf

---

### Phase 3: 每日简报 (Morning Brief)
**目标**: 每天早上自动推送

**方案**:
- [ ] 定时任务 (每天 7-8 点)
- [ ] 聚合: 天气 + 新闻 + 待办
- [ ] 推送到飞书

**技术栈**:
- 定时: OpenClaw Cron
- 推送: 飞书消息 API

---

## 🚀 实施顺序

```
Week 1: Phase 1 新闻聚合
  ↓
Week 2: Phase 2 知识库增强
  ↓
Week 3: Phase 3 每日简报
```
