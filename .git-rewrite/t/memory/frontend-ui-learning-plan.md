# 前端UI学习计划

## 学习阶段（5阶段）

### 第一阶段：HTML/CSS 基础 (2-4周)
- Week 1: HTML标签、语义化、SEO基础
- Week 2: CSS基础、选择器、盒模型
- Week 3: Flexbox 弹性布局
- Week 4: Grid 网格布局、响应式设计

### 第二阶段：JavaScript (4-6周)
- Week 5: 基础语法、数据类型
- Week 6: DOM操作、事件
- Week 7: 异步编程、API调用
- Week 8: ES6+新特性

### 第三阶段：UI/UX设计 (持续)
- Week 9: 设计原则（亲密性、对齐、重复、对比）
- Week 10: 设计系统、组件库
- Week 11: 动效设计、微交互
- Week 12: 用户体验优化

### 第四阶段：框架 (4-6周)
- Week 13-14: React基础
- Week 15-16: Vue基础
- Week 17-18: 状态管理、路由

### 第五阶段：实战作品集
- 完成3-5个实战项目
- 整理作品集网站
- 准备面试作品
