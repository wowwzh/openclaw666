### 练习1: NotificationBell 组件优化

**优化内容：**
- 添加 `useMemo` 优化 `unreadCount` 计算，避免不必要的重复计算
- 添加 `useCallback` 优化 `formatTime` 函数，避免函数重新创建

**优化前：**
```typescript
const unreadCount = notifications.filter(n => !n.read).length

const formatTime = (timestamp: number) => { ... }
```

**优化后：**
```typescript
const unreadCount = useMemo(() => 
  notifications.filter(n => !n.read).length, 
  [notifications]
)

const formatTime = useCallback((timestamp: number) => {
  // ...
}, [])
```

**效果：**
- 减少不必要的重渲染
- 当 notifications 数组引用变化时才重新计算
- 符合 React 性能最佳实践
