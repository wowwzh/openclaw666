# Watt Toolkit 自动化研究

## 研究目标
实现自动化调用 Watt Toolkit (Steam++) 来开启/关闭代理加速

## 已发现

### 1. 安装路径
```
C:\Program Files\WindowsApps\4651ED44255E.47979655102CE_3.0.1620.0_x64__k6txddmbb6c52\Steam++.exe
```

### 2. 进程信息
- Steam++.exe - 主程序 (MSIX 打包)
- Steam++.Accelerator.exe - 加速器模块

### 3. 代理端口
- 26561 - 代理监听端口
- 7890 - 备用端口
- 80/443 - 反向代理（已启用）

### 4. 问题
- **MSIX 应用无法直接运行** - 需要通过 Windows App SDK 或启动器
- **没有找到命令行参数** - --help 不工作
- **没有配置文件暴露代理开关**

## 研究结论

Watt Toolkit (Steam++) 是基于 .NET 的 MSIX 打包应用，可能没有提供自动化接口。

### 替代方案
1. **使用 Windows UI 自动化** - 通过模拟点击来开关代理
2. **使用系统代理设置** - 配置 Windows 系统代理
3. **寻找第三方 CLI 工具** - 可能社区有开发
