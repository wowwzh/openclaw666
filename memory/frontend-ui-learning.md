## 二、在线学习平台

### 1. freeCodeCamp (chinese.freecodecamp.org)
- **特点**：免费、交互式编程、动手实践
- **课程方向**：
  - 响应式网页设计
  - JavaScript算法和数据结构
  - 前端库（React等）
  - 数据可视化
  - API和微服务
  - 信息安全和质量保证
- **适合**：零基础入门，按部就班学习

### 2. The Odin Project (theodinproject.com) ⭐强烈推荐
- **特点**：开源免费、社区支持、强调动手能力
- **学习路径**：
  - **Foundations（基础）**：HTML、CSS、JavaScript入门
  - **Full Stack JavaScript（全栈JavaScript）**：
    1. 高级HTML/CSS
    2. JavaScript（DOM操作、面向对象编程、API）
    3. React（最流行的JavaScript UI库）
    4. 数据库（SQL、PostgreSQL）
    5. Express后端开发
    6. 职业发展指导
  - **Full Stack Ruby on Rails**
- **优势**：课程经过精心筛选，使用真实项目练习

### 3. Frontend Mentor (frontendmentor.io)
- **特点**：实战项目、真实设计稿、职业导向
- **核心价值**：
  - 设计稿到代码的实践
  - 构建作品集项目
  - AI-powered代码反馈
  - Figma设计文件
  - 社区支持
- **适合**：已有基础，想提升实战能力

### 4. 优优教程网 (uiiiuiii.com)
- **特点**：国内免费教程网站
- **内容**：设计教程、软件教程、设计灵感

### 5. 慕课网 (imooc.com)
- **特点**：国内系统课程平台
- **方向**：前端开发、UI设计、实战项目

## 四、推荐学习顺序

### 第一阶段：HTML & CSS (2-4周)
- 语义化HTML
- CSS选择器、盒模型
- Flexbox和Grid布局
- 响应式设计基础
- 练习：搭建静态页面

### 第二阶段：JavaScript (4-6周)
- 基础语法
- DOM操作
- 事件处理
- 面向对象编程
- Fetch API
- 练习：交互式组件

### 第三阶段：UI/UX设计 (持续学习)
- 阅读《设计心理学》
- 学习色彩理论
- 掌握排版技巧
- 理解用户体验要素
- 练习：设计系统

### 第四阶段：框架 (4-6周)
- React或Vue
- 组件化开发
- 状态管理
- 练习：构建完整应用

### 第五阶段：实战作品集 (持续)
- 使用Frontend Mentor项目
- 完成真实设计稿
- 构建个人作品集

## 八、学习建议

1. **动手优先**：看10遍不如做一遍
2. **项目驱动**：以完成项目为目标学习
3. **持续迭代**：定期回顾和重构代码
4. **关注设计**：技术之外培养设计感
5. **加入社区**：参与讨论，获取反馈
6. **记录成长**：写博客，记录学习历程
