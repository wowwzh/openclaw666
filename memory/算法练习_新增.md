## 合并两个有序数组 (LeetCode 88) ✅
- **题目**: 将两个有序数组合并到第一个数组
- **解法**: 从后向前遍历，避免元素移动
- **时间复杂度**: O(m+n)
- **空间复杂度**: O(1)
- **代码**:
```python
def merge(nums1, m, nums2, n):
    p1 = m - 1      # nums1有效元素末尾
    p2 = n - 1      # nums2末尾
    p = m + n - 1   # nums1末尾
    
    while p2 >= 0:
        if p1 >= 0 and nums1[p1] > nums2[p2]:
            nums1[p] = nums1[p1]
            p1 -= 1
        else:
            nums1[p] = nums2[p2]
            p2 -= 1
        p -= 1
```
- **思路**: 从后向前填充，避免覆盖未处理的元素
