### 1. 两数之和 (LeetCode 1) ✅
- **题目**: 给定数组和目标值，返回两数下标
- **解法**: 哈希表法
- **时间复杂度**: O(n)
- **空间复杂度**: O(n)
- **代码**:
```python
def twoSum(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        if target - num in seen:
            return [seen[target - num], i]
        seen[num] = i
    return []
```

### 2. 二叉树最大深度 (LeetCode 104) ✅
- **题目**: 求二叉树的最大深度
- **解法**: 递归 DFS
- **时间复杂度**: O(n)
- **空间复杂度**: O(h) h为树高
- **代码**:
```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def maxDepth(root):
    if not root:
        return 0
    return 1 + max(maxDepth(root.left), maxDepth(root.right))
```
- **思路**: 递归返回左右子树最大深度+1

### 3. 有效的括号 (LeetCode 20) ✅
- **题目**: 判断括号字符串是否有效
- **解法**: 栈 + 哈希表
- **时间复杂度**: O(n)
- **空间复杂度**: O(n)
- **代码**:
```python
def isValid(s):
    stack = []
    mapping = {')': '(', ']': '[', '}': '{'}
    
    for char in s:
        if char in mapping:
            if not stack or stack.pop() != mapping[char]:
                return False
        else:
            stack.append(char)
    
    return len(stack) == 0
```
- **思路**: 遇到左括号入栈，遇到右括号时检查栈顶是否匹配

### 4. 买卖股票的最佳时机 II (LeetCode 122) ✅
- **题目**: 给定数组表示每天股票价格，可多次买卖，求最大利润
- **解法**: 贪心算法
- **时间复杂度**: O(n)
- **空间复杂度**: O(1)
- **代码**:
```python
def maxProfit(prices):
    profit = 0
    for i in range(1, len(prices)):
        if prices[i] > prices[i-1]:
            profit += prices[i] - prices[i-1]
    return profit
```
- **思路**: 只要今天价格比昨天高，就昨天买入今天卖出（累加所有上涨区间）

### 5. 最大子数组和 (LeetCode 53) ✅
- **题目**: 找连续子数组的最大和
- **解法**: 贪心算法
- **时间复杂度**: O(n)
- **空间复杂度**: O(1)
- **代码**:
```python
def maxSubArray(nums):
    max_sum = nums[0]
    current_sum = nums[0]
    
    for i in range(1, len(nums)):
        current_sum = max(nums[i], current_sum + nums[i])
        max_sum = max(max_sum, current_sum)
    
    return max_sum
```
- **思路**: 遍历累加，如果当前和为负则舍弃，从下一个数重新开始

### 6. 反转链表 (LeetCode 206) ✅
- **题目**: 反转单链表
- **解法**: 迭代 (三指针)
- **时间复杂度**: O(n)
- **空间复杂度**: O(1)
- **代码**:
```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def reverseList(head):
    prev = None
    current = head
    
    while current:
        next_temp = current.next  # 保存下一个节点
        current.next = prev       # 反转指针
        prev = current           # 移动prev
        current = next_temp      # 移动current
    
    return prev
```
- **思路**: 用三个指针逐步反转链表节点指向

### 7. 旋转数组 (LeetCode 189) ✅
- **题目**: 将数组向右轮转k个位置
- **解法**: 反转法
- **时间复杂度**: O(n)
- **空间复杂度**: O(1)
- **代码**:
```python
def rotate(nums, k):
    k = k % len(nums)
    
    def reverse(l, r):
        while l < r:
            nums[l], nums[r] = nums[r], nums[l]
            l += 1
            r -= 1
    
    # 三次反转
    reverse(0, len(nums) - 1)  # 全部反转
    reverse(0, k - 1)           # 前k个反转
    reverse(k, len(nums) - 1)  # 后n-k个反转
```
- **思路**: [1,2,3,4,5,6,7] k=3 → 反转全部[7,6,5,4,3,2,1] → 反转前3[5,6,7,4,3,2,1] → 反转后4[5,6,7,1,2,3,4]

### 8. 合并两个有序数组 (LeetCode 88) ✅
- **题目**: 合并两个有序数组到第一个数组
- **解法**: 从后向前遍历
- **时间复杂度**: O(m+n)
- **空间复杂度**: O(1)
- **代码**:
```python
def merge(nums1, m, nums2, n):
    p1 = m - 1      # nums1有效元素末尾
    p2 = n - 1      # nums2末尾
    p = m + n - 1   # 合并后位置
    
    while p2 >= 0:
        if p1 >= 0 and nums1[p1] > nums2[p2]:
            nums1[p] = nums1[p1]
            p1 -= 1
        else:
            nums1[p] = nums2[p2]
            p2 -= 1
        p -= 1
```
- **思路**: 从后向前比较，避免覆盖nums1的有效元素

### 9. 盛最多水的容器 (LeetCode 11) ✅
- **题目**: 给定height数组代表墙壁高度，找两根柱子能盛最多水
- **解法**: 双指针（从两端向中间逼近）
- **时间复杂度**: O(n)
- **空间复杂度**: O(1)
- **代码**:
```python
def maxArea(height):
    left = 0
    right = len(height) - 1
    max_water = 0
    
    while left < right:
        # 计算当前面积
        width = right - left
        h = min(height[left], height[right])
        max_water = max(max_water, width * h)
        
        # 移动较短的那边（因为移动较长的不会让面积增大）
        if height[left] < height[right]:
            left += 1
        else:
            right -= 1
    
    return max_water
```
- **思路**: 从两端开始，移动较短的柱子可能找到更高的柱子来增大面积

### 10. 每日温度 (LeetCode 739) ✅
- **题目**: 给定温度数组，返回每一天需要等多少天才能等到更暖和的温度
- **解法**: 单调栈（存储下标）
- **时间复杂度**: O(n)
- **空间复杂度**: O(n)
- **代码**:
```python
def dailyTemperatures(temperatures):
    n = len(temperatures)
    answer = [0] * n
    stack = []  # 存储下标，温度单调递增
    
    for i in range(n):
        while stack and temperatures[i] > temperatures[stack[-1]]:
            prev_index = stack.pop()
            answer[prev_index] = i - prev_index
        stack.append(i)
    
    return answer
```
- **思路**: 用栈维护递增温度，遇到更高温度时弹出并计算天数差

### 11. 字符串相乘 (LeetCode 43) ✅
- **题目**: 给定两个字符串形式的非负整数，返回它们的乘积
- **解法**: 模拟竖式乘法
- **时间复杂度**: O(m*n)
- **空间复杂度**: O(m+n)

```python
def multiply(num1, num2):
    if num1 == "0" or num2 == "0":
        return "0"
    
    m, n = len(num1), len(num2)
    result = [0] * (m + n)
    
    for i in range(m - 1, -1, -1):
        for j in range(n - 1, -1, -1):
            mul = int(num1[i]) * int(num2[j])
            p1, p2 = i + j, i + j + 1
            sum_val = mul + result[p2]
            result[p1] += sum_val // 10
            result[p2] = sum_val % 10
    
    result_str = ''.join(map(str, result))
    return result_str.lstrip('0') or "0"
```

### 14. 有效的字母异位词 (LeetCode 242) ✅
- **题目**: 判断两个字符串是否为字母异位词
- **解法**: 字符计数 O(n)
- **时间复杂度**: O(n)
- **空间复杂度**: O(1) 或 O(k) 其中k为字符集大小
- **代码**:
```python
def isAnagram(s, t):
    if len(s) != len(t):
        return False
    count = {}
    for c in s:
        count[c] = count.get(c, 0) + 1
    for c in t:
        count[c] = count.get(c, 0) - 1
        if count[c] < 0:
            return False
    return True
```
- **思路**: 遍历第一个字符串计数，遍历第二个字符串减计数，如果出现负数则不是异位词

- **时间复杂度**: O(n!*n)

```python
def permute(nums):
    result = []
    
    def backtrack(start):
        if start == len(nums):
            result.append(nums[:])
            return
        for i in range(start, len(nums)):
            nums[start], nums[i] = nums[i], nums[start]
            backtrack(start + 1)
            nums[start], nums[i] = nums[i], nums[start]
    
    backtrack(0)
    return result
```
