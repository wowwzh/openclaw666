## 新技能: config-backup

### 备份恢复系统设计 (Kimi 生成)

**系统架构:**
- 配置监控模块 → 配置校验 → 自动恢复 → Gateway重启

**5大核心模块:**
1. 配置监控 - 监控配置文件变化
2. 配置验证 - 检查必填字段、数据类型、业务逻辑
3. 备份管理 - 自动备份、版本管理、Hash校验
4. 自动恢复 - 检测错误、自动回滚
5. Gateway重启 - 检测状态、自动启动

**需要备份的关键文件:**
- Gateway配置: ~/.openclaw/config.json
- 技能配置: workspace/skills/*/SKILL.md
- 记忆文件: workspace/memory/*.md

**恢复触发条件:**
- 配置文件解析失败
- 必填字段缺失
- 数值范围错误
- Gateway 无响应

**注意事项:**
- 自动重启 Gateway 需要哥哥授权
- 备份文件保留最近 5 个版本
