# 国内AI学习平台汇总

## 一、主流AI学习平台

### 1. 飞桨AI Studio（百度）
- 网址: https://aistudio.baidu.com
- 定位: 国内领先的AI学习与实训社区
- 特色: 基于PaddlePaddle框架，提供免费GPU算力和Jupyter Notebook环境
- 内容: 深度学习、计算机视觉、自然语言处理、强化学习等课程

### 2. 慕课网（IMOOC）
- 网址: https://www.imooc.com
- 定位: 专注IT与AI技能实战的在线教育平台
- 规模: 1800名一线互联网企业专家讲师，3000+门课程，2500万+用户
- AI课程: 117种AI相关课程

### 3. 阿里云AI学习路线
- 特色: 30门在线课程+22个实战案例
- 内容: 机器学习、深度学习、NLP
- 实践: 10分钟快速搭建网站AI助手

### 4. 华为云开发者学堂
- 课程: 《AI全栈成长计划》系列
- 覆盖: AI基础知识、图像分类、自然语言处理

### 5. 扣子平台（Coze）
- 定位: AI助手可视化搭建平台
- 特色: 无需编码，10分钟搭建专属AI助手
- 功能: 插件集成、对话流程设计、多模型配置
